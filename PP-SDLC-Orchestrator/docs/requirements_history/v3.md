# Requirement Baseline — Version 3: Conversational Application

> Preserved verbatim as design history. Do not delete or silently edit — supersede only via a new ADR referencing this baseline.

The primary interface is a simple, ChatGPT/Claude-style conversation experience combined with structured controls for:

- lifecycle
- artefacts
- versions
- reviews
- rework
- approvals
- clarifications
- assumptions
- risks
- decisions
- activity history
- standalone agents
