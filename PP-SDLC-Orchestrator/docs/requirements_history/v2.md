# Requirement Baseline — Version 2: Pluggable Agents

> Preserved verbatim as design history. Do not delete or silently edit — supersede only via a new ADR referencing this baseline.

Every specialist agent must work in:

- `ORCHESTRATED` mode
- `STANDALONE` mode

All agents must use a common contract, stable inputs and outputs, strict domain boundaries, versioning, input-sufficiency rules, handoffs, review states, and project isolation.
