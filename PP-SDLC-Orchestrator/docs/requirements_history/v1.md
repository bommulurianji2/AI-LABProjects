# Requirement Baseline — Version 1: Frozen MVP

> Preserved verbatim as design history. Do not delete or silently edit — supersede only via a new ADR referencing this baseline.

The original frozen model contains 11 agents:

1. SDLC Orchestrator Agent
2. Analysis Agent
3. UX Design Agent
4. Technical Design Agent
5. Data & Integration Agent
6. Governance & Security Agent
7. Build Agent
8. Validation / QA Agent
9. Test Agent
10. Deploy Agent
11. Hypercare & Closure Agent

## Rules

- Mandatory human review after every lifecycle phase.
- Every rerun generates a new artefact version.
- Approved artefacts must never be silently overwritten.
- Rework and backtracking must be routed through the Orchestrator.
- Specialist Agent Skills own Power Platform and M365 domain guardrails.
- The Orchestrator owns project state, execution, approvals, routing, exceptions, and version tracking.
- Agents maintain domain reference material.
- Build maturity Levels 1 and 2 are included.
- Level 3 is optional and controlled.
- Level 4 permits Power Platform CLI integration.
- Power Platform delivery pipelines remain separately controlled.
- The interactive HTML prototype remains HTML.
